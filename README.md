# 暖邻帮 - UniApp跨平台小程序/App

为老年人和残障人士提供信息聚合与服务对接的互助平台。

## 项目概述

「暖邻帮」是一款面向老年人和残障人士的社区互助服务小程序/App，旨在通过整合社区资源，为特殊群体提供便捷的信息查询和服务对接服务。

## 核心功能

### 1. 信息聚合
- **政务服务**: 政策解读、办事指南、政务通知
- **生活服务**: 家政、维修、购物、餐饮配送
- **出行服务**: 轮椅陪同、无障碍导航、公交信息
- **医疗健康**: 就医陪同、健康咨询、用药提醒

### 2. 服务对接
- 服务发布: 志愿者和服务提供者可发布服务
- 智能匹配: 基于位置和需求的服务匹配
- 在线沟通: 即时通讯功能
- 订单管理: 完整的订单流转系统

### 3. 无障碍功能
- **语音识别**: 点击语音输入,智能识别需求
- **语音播报**: 文字转语音,页面内容朗读
- **大字体**: 多档位字体大小调节
- **高对比度**: 适合视障用户的高对比模式
- **无障碍地图**: 标记无障碍设施的地图服务

## 技术栈

- **框架**: UniApp (基于Vue3 + TypeScript)
- **UI组件**: TDesign (腾讯企业级设计体系)
- **状态管理**: Pinia
- **地图服务**: 腾讯地图SDK
- **语音识别**: 微信语音识别插件
- **数据存储**: LocalStorage + 微信云开发(可选)

## 开发指南

### 环境准备

1. 安装Node.js (v16+)
2. 安装HBuilderX编辑器
3. 下载微信开发者工具

### 安装依赖

```bash
npm install
```

### 运行项目

#### 微信小程序
```bash
npm run dev:mp-weixin
```
然后在微信开发者工具中导入 `dist/dev/mp-weixin` 目录

#### H5
```bash
npm run dev:h5
```

### API配置

在 `utils/config.ts` 中配置以下API密钥:

- `QQMAP_CONFIG.key`: 腾讯地图密钥
- `WEATHER_CONFIG.key`: 天气API密钥
- `VOICE_CONFIG`: 语音识别配置

申请地址:
- 腾讯地图: https://lbs.qq.com/console/key.html
- 高德天气: https://console.amap.com/dev/key/app
- 微信语音插件: https://developers.weixin.qq.com/miniprogram/dev/plugin/

## 使用说明

### 用户流程

1. **注册登录**: 微信授权或手机号验证码登录
2. **选择身份**: 老年人 / 残障人士 / 志愿者
3. **浏览服务**: 在服务大厅浏览或语音搜索
4. **下单预约**: 选择服务,填写需求,确认订单
5. **服务对接**: 与服务提供者在线沟通
6. **完成评价**: 服务完成后进行评价

### 志愿者流程

1. **注册登录**: 选择志愿者身份
2. **发布服务**: 填写服务详情,设置价格和服务范围
3. **接收订单**: 查看订单列表,确认服务
4. **提供服务**: 按约定提供服务
5. **获得评价**: 用户评价后积累信用

## 无障碍设计

### WCAG 2.1标准遵循
- 文字替代(Alt属性)
- 键盘可访问性
- 充足对比度
- 焦点可见性

### 针对老年用户
- 大字体(最大支持1.5倍)
- 高对比度模式
- 简洁清晰的界面
- 重要信息突出显示
- 减少操作步骤

### 针对视障用户
- 语音播报功能
- 屏幕阅读器兼容
- 高对比度色彩
- 简化的导航结构
